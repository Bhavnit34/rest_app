### Installation
Install dependencies
```sh
$ npm install
```
Start nodeJS server
```sh
$ node bin/www
```
Verify the deployment by navigating to your server address in your preferred browser.
```sh
127.0.0.1:3000
```

### Command Line Arguments
| Arg | options |
| ------ | ------ |
|--logLevel | silly, debug, verbose, **info**, warn, error|
|--logPath | **/logs** |
